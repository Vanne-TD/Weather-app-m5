Sobre esta aplicación

Una herramienta creada para entregar información meteorológica clara, accesible y útil para las principales localidades de la Región del Biobío.

Esta aplicación fue desarrollada por Vanessa Torres Drake como parte del pre‑portafolio del Bootcamp de Frontend de Talento Digital para Chile.

Su objetivo es entregar información precisa y actualizada sobre el clima en 12 localidades de la Región del Biobío, permitiendo a los usuarios planificar sus actividades diarias y semanales de manera más eficiente.

Buenas noticias!!! ya contamos con datos reales de las principales localidades, ya que hemos conectado una API meterológica que nos proporciona datos en tiempo real, actualizar automáticamente las temperaturas, íconos y pronósticos, y ofrecer una experiencia aún más completa.

¡Espero que disfrutes esta aplicación y que pronto puedas consultar tu clima de forma rápida, sencilla y confiable!

🎯 Objetivo del Proyecto

Crear una plataforma simple, visual y responsiva que permita consultar el clima actual y el pronóstico semanal de distintas localidades del Biobío.

⚙️ Funcionalidades

  • Clima actual por localidad.
  • Pronóstico semanal con íconos representativos.
  • Página de detalles con información ampliada.
  • Datos en tiempo real mediante uso de API meteorológica.
  • Navegación clara y diseño responsivo.

💻 Tecnologías Utilizadas

  • HTML5 semántico
  • SCSS con metodología BEM
  • Bootstrap 5
  • JavaScript ES6+
  • Programación Orientada a Objetos (POO)
  • Consumo de API de clima openweathermap.org (fetch + async/await)
  • DOM dinámico para Home y Detalle
